# 🏀 Parabola Extender (포물선 연장기)

Parabola Extender는 웹 브라우저나 데스크탑 화면에 나타나는 미완성된 점선 포물선을 실시간으로 감지하여, 이를 완벽한 기하학적 포물선으로 연장해 투명 오버레이로 그려주는 Python 애플리케이션입니다. 특히 농구 게임과 같이 궤적 예측이 필요한 환경에 최적화되어 있습니다.

## ✨ 주요 기능

- **실시간 인식 모드 (Detection Mode)**: 
  - **Top-Hat 변환**: 배경색(흰색, 파란색 등)에 방해받지 않고 "검정 테두리가 있는 흰색 점"을 정확히 추출합니다.
  - **가상 궤적 인식**: 듬성듬성한 점들을 이은 가상의 포물선을 계산하여 매끄러운 실선으로 연장합니다.
  - **동적 발사점**: 좌측 시작점은 고정되지 않고 감지된 점들의 흐름을 자연스럽게 따라갑니다.
- **고정 모드 (Fixed Mode)**: 
  - **자동 타겟팅**: 골대 림(Rim)의 정중앙과 농구공의 중심을 실시간으로 탐지하여 연결합니다.
  - **최적의 높은 포물선**: 두 지점 사이의 거리를 계산하여 이상적인 높은 궤적(High Arc)을 자동으로 생성합니다.
- **정밀한 객체 탐지**: 
  - **원형도(Circularity) 분석**: 빨간색 골대 중 실제 공이 통과해야 할 '림' 부분의 중앙을 정확히 포착합니다.
  - **영역 필터링**: 좌상단 점수판 아이콘 등 UI 요소를 공으로 오인하지 않도록 지능적으로 배제합니다.
- **투명 네온 오버레이**: PyQt5를 이용해 마우스 클릭을 통과시키며 항상 최상위를 유지하는 투명 창을 제공합니다.
- **컨트롤 패널**: FPS, 연장 길이, 평활화, 감지 임계값 등을 실시간으로 조절할 수 있습니다.

## 🛠 기술 스택

- **언어**: Python 3.10+
- **GUI & Overlay**: PyQt5
- **이미지 처리**: OpenCV (OpenCV-Python)
- **수치 계산**: NumPy
- **화면 캡처**: mss
- **시스템 모니터링**: psutil

## 🚀 설치 및 실행 방법

### 1. 의존성 설치
```bash
pip install -r requirements.txt
```

### 2. 앱 실행
```bash
python main.py
```

## 📖 사용 방법

1. **영역 선택**: `Select Region` 버튼을 누르고, 골대와 공이 나타나는 화면 영역을 드래그하여 지정합니다.
2. **시작**: `Start ▶` 버튼을 누르면 오버레이가 활성화됩니다.
3. **모드 전환**:
   - `Detection Mode`: 실제 화면에 나타나는 점선을 추적하여 연장합니다.
   - `Fixed Mode`: 골대 중앙과 공을 잇는 높은 포물선 가이드를 즉시 생성합니다.
4. **설정 조절**:
   - **Extension %**: 포물선을 좌우로 얼마나 더 길게 그릴지 조절합니다.
   - **Threshold**: 배경 대조에 따라 점과 객체를 더 잘 찾도록 민감도를 조절합니다.

## 📂 프로젝트 구조

```text
parabola_extender/
├── main.py           # 애플리케이션 진입점 및 조정 레이어
├── capture.py        # QThread 기반 실시간 캡처 루프
├── detector.py       # OpenCV 기반 객체 및 림(Rim) 중앙 감지
├── fitter.py         # 2차 곡선 회귀 및 고정 포물선 생성
├── overlay.py        # 투명 오버레이 및 네온 실선 렌더링
├── controls.py       # 모드 전환 및 사용자 컨트롤 패널
└── requirements.txt  # 의존성 라이브러리 목록
```

## ⚠️ 주의 사항 (macOS)
- 시스템 설정의 **[보안 및 개인정보 보호 > 화면 기록]** 권한이 허용되어 있어야 합니다.
- macOS의 Space 전환 문제를 피하기 위해 좌표 기반 오버레이 방식을 사용합니다.
