# 🏀 Parabola Extender (포물선 연장기)

Parabola Extender는 웹 브라우저나 데스크탑 화면에 나타나는 미완성된 점선 포물선을 실시간으로 감지하여, 이를 완벽한 기하학적 포물선으로 연장해 투명 오버레이로 그려주는 Python 애플리케이션입니다.

> **[Latest Update]** 최근 업데이트를 통해 골대와 공을 자동으로 연결하는 **'고정 모드(Fixed Mode)'**가 추가되었습니다.

## ✨ 주요 기능

- **🚀 [NEW] 고정 모드 (Fixed Mode)**: 
  - **자동 타겟팅**: 빨간색 골대 림(Rim)의 정중앙과 농구공의 중심을 실시간으로 탐지하여 연결합니다.
  - **최적의 포물선**: 두 지점 사이의 거리를 계산하여 가장 이상적이고 완만한 낮은 궤적을 자동으로 생성합니다.
  - **간편 토글**: 컨트롤 패널의 버튼 하나로 실시간 인식 모드와 고정 모드를 자유롭게 오갈 수 있습니다.

- **실시간 인식 모드 (Detection Mode)**: 
  - **Top-Hat 변환**: 배경색에 상관없이 "검정 테두리가 있는 흰색 점"을 정확히 추출합니다.
  - **가상 궤적 인식**: 듬성듬성한 점들을 이은 가상의 포물선을 계산하여 매끄러운 실선으로 연장합니다.
  - **동적 시작점**: 감지된 점들의 흐름을 자연스럽게 따라가며 실시간으로 대응합니다.

- **정밀한 객체 탐지**: 
  - **원형도(Circularity) 분석**: 골대 림 부분을 정확히 포착하여 기둥이나 백보드가 아닌 '중앙'에 고정합니다.
  - **지능형 필터링**: 화면 좌상단 점수판 아이콘 등 UI 요소를 감지 대상에서 자동으로 배제합니다.

- **투명 네온 오버레이**: PyQt5를 이용해 마우스 클릭을 통과시키며 항상 최상위를 유지하는 투명 창을 제공합니다.

- **컨트롤 패널**: FPS, 연장 길이, 평활화, 감지 임계값 등을 실시간으로 조절할 수 있습니다.

## 🛠 기술 스택

- **언어**: Python 3.10+
- **GUI & Overlay**: PyQt5
- **이미지 처리**: OpenCV (OpenCV-Python)
- **수치 계산**: NumPy
- **화면 캡처**: mss

## 🚀 설치 및 실행 방법

### 1. 의존성 설치
```bash
pip install -r requirements.txt
```

### 2. 앱 실행
```bash
python main.py
```

## 📖 사용 방법

1. **영역 선택**: `Select Region` 버튼을 누르고, 골대와 공이 나타나는 화면 영역을 드래그하여 지정합니다.
2. **시작**: `Start ▶` 버튼을 누르면 오버레이가 활성화됩니다.
3. **모드 전환**:
   - `Detection Mode`: 실제 화면에 나타나는 점선을 추적하여 연장합니다.
   - `Fixed Mode`: 골대 중앙과 공을 잇는 가이드 곡선을 즉시 생성합니다.
